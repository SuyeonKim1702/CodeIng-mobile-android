# CodeIng-mobile 
- 코딩 종합 플랫폼 코드잉의 모바일 버전 (beta)
- Nativescript + Angular를 이용한 개발 

### 주요 기능 
- 강의 검색 
- 맞춤형 강의 추천 
- 클래스룸 
- 강의 랭킹
